## 팀미션 1 (20252410 정승우)

**실현된 기술**

- 전기차

**느낀점**

기술에 대한 정보를 많이 알고 있어야 그럴듯한 근거 있는 미래를 상상할 수 있구나를 느꼈고,  
과학기술은 실생활 편의에 관련된 것이 정말 빠르게 변화하는구나를 체감했습니다.  
그래서 앞으로의 미래를 예측하기 위해서는 기술 정보에 민감해져야겠다고 생각이 들었습니다.
